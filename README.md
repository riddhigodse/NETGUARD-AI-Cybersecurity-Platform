# NETGUARD AI - Cybersecurity Platform

A Java-based real-time Deep Packet Inspection and Network Traffic Monitoring Dashboard that captures live packets, analyzes traffic, detects websites through SNI inspection, displays protocol statistics, and identifies suspicious traffic activity using packet analysis techniques.

## Features

* Real-Time Packet Capture
* Deep Packet Inspection (DPI)
* Website Detection using SNI Analysis
* Live Network Traffic Monitoring
* Suspicious Traffic Detection
* Cybersecurity Alerts System
* Interactive Analytics Dashboard
* AI Security Assistant
* CSV Report Generation
* Modern JavaFX User Interface

## Technologies Used

* Java
* JavaFX
* Maven
* Pcap4J
* CSV Processing
* Network Packet Analysis

## Project Modules

### Dashboard

Displays network statistics and monitoring overview.

### Live Capture

Captures and analyzes network packets in real time.

### Alerts

Detects suspicious traffic and security events.

### Reports

Exports captured traffic data into CSV reports.

### Analytics

Visualizes network activity through charts and statistics.

### NETGUARD AI

Provides cybersecurity guidance and threat analysis.

### Settings

Displays platform status and security engine information.
